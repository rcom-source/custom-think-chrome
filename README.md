# Claude Custom Think — Chrome Extension

Custom thinking indicators for Claude on the web (claude.ai and claude.com). Replace the pulsing animation with themed rotating words.

## What it does

When Claude is thinking, you'll see rotating words from your chosen category:

```
Fetching...      (Dogs)
Launching...     (Space)
Brewing...       (Coffee Shop)
Cogitating...    (Oxford English Dictionary)
Breathing in...  (Zen)
```

## Install

### From source (developer mode)

1. Clone this repo:
   ```bash
   git clone https://github.com/rcom-source/custom-think-chrome.git
   ```
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the cloned directory

## Usage

Click the extension icon in Chrome to:

- Pick a word category (Cats, Dogs, Space, Coffee, etc.)
- Toggle the extension on/off
- Switch between inline and floating pill display
- Choose session-only or permanent preferences
